Tableau de bord

La page d'accueil après connexion. Elle offre une vue d'ensemble en temps réel de tous vos projets avec des indicateurs clés de performance.

👥
Visibilité selon le rôle : Les Study Directors ne voient que leurs projets. Les Project Managers voient uniquement les projets dont ils font partie. Les autres rôles voient tous les projets.

📈 Indicateurs KPI

Quatre cartes en haut du tableau de bord affichent les métriques globales :

🟢

Projets en cours

Nombre de projets au statut in progress.

🔴

Non-conformités ouvertes

Nombre total de constatations QA non résolues.

⏳

Inspections QA en attente

Inspections planifiées mais pas encore réalisées.

📊

Achèvement moyen

Pourcentage moyen de complétion de tous les projets visibles.

🔍 Filtres par statut

Des liens de filtre permettent d'afficher uniquement les projets d'un statut donné :

FiltreDescription
TousAffiche tous les projets (par défaut)
In ProgressProjets en cours d'exécution
Not StartedProjets créés mais pas encore démarrés
SuspendedProjets temporairement suspendus
CompletedProjets terminés (toutes phases complètes)
ArchivedProjets archivés, en lecture seule

🗂️ Cartes de projets

Chaque projet est affiché sous forme de carte contenant :

  • Code et titre du projet
  • Barre latérale colorée indiquant le statut (vert = in progress, gris = archivé, etc.)
  • Informations d'équipe : Study Director, Project Manager
  • Métriques de complétion : activités, inspections, constatations, documents
  • Score global (pourcentage d'avancement)
  • Bouton Modifier super_adminfacility_manager
💡
Cliquer sur le titre ou le code d'un projet vous amène à la vue détaillée du projet (Project Overview) qui récapitule toutes les phases, inspections et documents.

📄 Pagination

Les projets sont affichés par groupes de 6 par page. Utilisez les boutons de pagination en bas de la liste pour naviguer.
