Planning & Master Schedule

Outils de visualisation et de suivi des activités à l'échelle d'un projet ou de tous les projets simultanément, avec exports PDF.

👥
Le Master Schedule est accessible à tous les utilisateurs connectés. Le Project Tracking Sheet est également ouvert à tous.

🗓️ Master Schedule

Le Master Schedule (/master-schedule) offre une vue consolidée de tous les projets et de leurs activités dans le temps.

Contenu affiché

  • Tous les projets actifs avec leur timeline
  • Activités planifiées avec dates estimées et réelles
  • Statut de chaque activité par code couleur
  • Phases critiques mises en évidence
  • Jalons importants (réunions, inspections)

Export PDF

Le bouton Télécharger PDF génère une version imprimable du Master Schedule via /master-schedule/pdf.

  • Format paysage pour une meilleure lisibilité
  • En-tête avec date de génération
  • Tous les projets et activités inclus

📋 Project Tracking Sheet

Le Project Tracking Sheet (/project-activity-schedule) permet de suivre les activités d'un projet spécifique.

Filtrer par projet

  1. Accédez au Project Tracking Sheet depuis la navigation
  2. Sélectionnez un projet dans le menu déroulant
  3. La page se recharge avec les activités du projet sélectionné

Données affichées

  • Liste complète des activités du projet avec leurs catégories
  • Responsable de chaque activité
  • Date estimée vs date réelle
  • Statut : pending / in_progress / completed / delayed
  • Indicateur de phase critique

Modifier le suivi

Le formulaire de suivi permet de mettre à jour les informations d'avancement :

  1. Modifiez les dates ou statuts dans le tableau
  2. Cliquez sur Enregistrer pour sauvegarder via saveProjectTrackingSheet

Export PDF du projet

Le bouton Exporter PDF génère via /project/{id}/activities/pdf un document comprenant :

  • En-tête avec code et titre du projet
  • Toutes les activités avec statuts et dates
  • Phases critiques identifiées

🎨 Codes de statut des activités

StatutSignificationCouleur
pending En attente — pas encore démarrée Gris
in_progress En cours d'exécution Bleu
completed Complétée avec succès Vert
delayed En retard par rapport à la date estimée Rouge

📅 Réunions

Réunion d'initiation de l'étude

Planifiée depuis la Phase 4 (Planning) :

  1. Cliquez sur Planifier la réunion d'initiation
  2. Renseignez date, heure, lieu et participants
  3. Cliquez sur Enregistrer
  4. Un e-mail de notification est envoyé automatiquement aux participants

Réunion de phase critique

Planifiable pour chaque activité marquée comme phase critique :

  • Accéder au modal via le bouton Planifier réunion critique sur l'activité
  • Renseigner les informations de la réunion
  • Supprimer une réunion si annulée

Consulter les informations d'une réunion

Cliquer sur le titre de la réunion affiche les détails via l'endpoint AJAX /ajax/get-meeting-info.
